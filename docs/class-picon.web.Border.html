Class Border

A border works in the same way as a panel but does ont replace the origonal content of the HTML tag it is added to.

The border mark-up:

<picon:border> <h2>border heading</h2> <picon:body /> <p>other border content</p> </picon:border>

And in the composing class:

<h1>heading</h1> <div picon:id="myBorder"> Border content will wrap this content </div> <p>other mark-up...</p>

Would be rendered as:

<h1>heading</h1> <div picon:id="myBorder"> <h2>border heading</h2> Border content will wrap this content <p>other border content</p> </div> <p>other mark-up...</p>
