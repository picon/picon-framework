Class PageMap

Holder for the map to all statfull and statless web pages

A singleton which will persist from one request to another via the session For this reason it is important that all stored objects are serializable

The page map stores a map of all stateless pages by both their direct path (the name of the class, including its namespace) and also all specified paths (pages which have been given @Path) Pagges are expected to extend WebPage @see WebPage

Additionally, actual instances of pages which have deemed statfull are stored against an associated page id.
