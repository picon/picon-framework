Panels

A panel allows for a block of mark-up to be encapsulated into reusable components. A panel consists of an HTML file and a class containing the code to drive it. Panel can be rendered on any HTML tag.


The panel mark-up:

<picon:panel>
<h2>panel heading</h2>
</picon:panel>

And in the composing class:

<h1>heading</h1>
<div picon:id="myPanel">
Panel content will appear here
</div>
<p>other mark-up...</p>

Would be rendered as:

<h1>heading</h1>
<div picon:id="myPanel">
<h2>panel heading</h2>
</div>
<p>other mark-up...</p>
