Markup Inheritance

Picon enables mark-up to be inherited in the same way as methods from parent classes. This is a useful way of creating global page layouts. For example, the MarkupInheritancePage class extends the AbstractPage class. Both classes have associated mark-up and with some special Picon mark-up tags, the two mark-up sources can be merged.


In the child class:

<picon:extend>
<h2>Sub class heading</h2>
</picon:extend>

And in the parent class:

<h1>Parent class heading</h1>
<picon:child />
<p>Rest of parent markup....</p>

Would be merged and rendered as:

<h1>Parent class heading</h1>
<h2>Sub class heading</h2>
<p>Rest of parent markup....</p>
